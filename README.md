Immunoinformatic prediction of immunogenic polypeptides acroos the entire proteome of Giardia lamblia
* This repository contains all dataset related to the prediction of immunogenic polypeptides acroos the entire proteome of assemblages A and B of Giardia lamblia

Files description

*Analysis_assemblagesB_GS: This folder contains all data of the prediction by using different thresholds acroos the entire proteome of Giardia lamblia assemblage B(GS). File "0.01" saves predictions using a threshold of percentile rank <0.01 and IC50<50 nM. File "0.1" saves predictions using a threshold of percentile rank <0.1 and IC50<50 nM. File "1" saves predictions using a threshold of percentile rank <1 and IC50<50 nM

*Analysis_assemblagesA_WB: This folder contains all data of the prediction by using different thresholds acroos the entire proteome of Giardia lamblia assemblage A(WB). File "0.01" saves predictions using a threshold of percentile rank <0.01 and IC50<50 nM. File "0.1" saves predictions using a threshold of percentile rank <0.1 and IC50<50 nM. File "1" saves predictions using a threshold of percentile rank <1 and IC50<50 nM
